The SLOT Project

SLOT

The Supervised Learning Opportunities By Text project matches medical students to in hospital learning opportunies via text messaging.

Edinburgh Pilot

Learn More Below

Background

The SLOT project started at an NHS hackday. Sometimes junior doctors have the time and opportunity to supervise a medical student through a procedure, but there are none in close proximity. SLOT informs medical students about these opportunities and lets them claim them.


Open Source

The SLOT Project is entirely open source, and you can view all of the code that runs the project at github.com
